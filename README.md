# CRUD API + FastAPI

Este es un sistema (API) de seguros con FastAPI y SQLite3 usando SQLAlchemy

# Estructura del proyecto

sistema-seguros/
├── app/
│   ├── __init__.py
│   ├── main.py
│   ├── database.py
│   ├── models.py
│   ├── schemas.py
│   ├── routers/
│   │   ├── __init__.py
│   │   ├── customers.py
│   │   ├── policies.py
│   │   ├── assets.py
│   │   └── payments.py
│   └── config.py
├── requirements.txt
├── .env
└── .gitignore


# Comandos 

Instalar todas las dependencias

```shell
$ .env/Scripts/active.bat # (En Windows)
$ pip install -r requirements.txt
```
